void Server::sendData ( args  )

Sets the data or properties of plotHandle according the values stored in dataDict. Multiple objects can be referenced using Python-style pretty indexing (see below).

For a single set of data, dataList should be a 1D list. For a multiple sets of data destined for multiple axes or plots, dataList should be a 2D list, regardless of the number of data points. Where the first dimension contains a list for each dataset. In the following example, a figure has an axis group of 3 (so they have the same handle). Each axis contains a line plot with the handle "myData":

[ "sendData", "myFigure.axes[0:3].myData", {"x": [0,1,2], "y": [ [2,4,1], [9,1,0], [1,2,3] ]} ]

Parameters
reqRequest: ["sendData", handle, dataDict, parameter (optional)]
handleThe hierarchical handle to the plot of interest (e.g. myFigure.axis[0].experiment1Data)
dataDictThe data dictionary should contain one or both of "x" and "y" data for a 2D plot (e.g. {"x": [0,1,2], "y": [2,4,1]})
parameterOptional argument which informs the server which changing parameter that this update is in response to. See Socket Architecture for details
Returns
Success: [0, {"message": m}]
